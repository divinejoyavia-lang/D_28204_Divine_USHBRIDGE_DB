## Project Overview
This project is a database and PL/SQL implementation for **USH BRIDGE Consults**, a company specializing in architectural design and visual & graphic branding services.

The system manages:
- Clients
- Projects
- Designs
- Sales & payments
- Team members (architects, designers, consultants)

It also enforces business rules, logs activities, and supports analytics and business intelligence reporting.

## Technologies Used
- Oracle Database (Multitenant – PDB)
- SQL & PL/SQL
- Oracle SQL Developer
- GitHub for version control

## Project Phases Implemented
- Phase I – Problem definition & context
- Phase II – System design
- Phase III – ER model, normalization & data dictionary
- Phase IV – Database creation (PDB)
- Phase V – Tables, constraints & relationships
- Phase VI – Procedures, functions & packages
- Phase VII – Triggers, audit logging & restrictions
- Phase VIII – Business Intelligence & documentation
