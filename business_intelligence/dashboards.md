# Dashboard Mockups and Descriptions

## 1. Executive Summary Dashboard
**Audience:** Company management

**Key Components:**
- Total number of projects
- Total revenue generated
- Number of active and completed projects
- Top 5 highest revenue projects

**Purpose:**
Provides a quick overview of business performance and financial health.

---

## 2. Project Performance Dashboard
**Audience:** Project managers

**Key Components:**
- Projects by status (Pending, Active, Completed)
- Project duration comparison
- Budget vs actual revenue
- Project ranking by revenue

**Purpose:**
Helps project managers monitor progress and resource allocation.

---

## 3. Audit & Security Dashboard
**Audience:** System administrators

**Key Components:**
- Total database actions
- Number of denied actions
- Actions by type (INSERT, UPDATE, DELETE)
- Actions by user

**Purpose:**
Monitors security, policy enforcement, and suspicious activity.
