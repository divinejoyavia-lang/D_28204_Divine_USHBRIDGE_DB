# Key Performance Indicators (KPI) Definitions

## KPI 1: Total Revenue per Project
- **Description:** Total amount of money generated by each project.
- **Calculation:** SUM(amount) from the Sales table grouped by project.
- **Business Value:** Helps management identify profitable projects.

## KPI 2: Project Completion Rate
- **Description:** Percentage of projects marked as completed.
- **Calculation:** (Completed projects / Total projects) Ã— 100.
- **Business Value:** Measures delivery efficiency.

## KPI 3: Active Projects Count
- **Description:** Number of ongoing projects.
- **Calculation:** COUNT projects where project_status = 'Active'.
- **Business Value:** Indicates current workload.

## KPI 4: Denied Database Actions
- **Description:** Number of blocked INSERT, UPDATE, or DELETE actions.
- **Calculation:** COUNT records in audit_log where action_status = 'DENIED'.
- **Business Value:** Monitors security and policy enforcement.

## KPI 5: Revenue Trend Over Time
- **Description:** Changes in revenue across time.
- **Calculation:** Running total using window functions on Sales data.
- **Business Value:** Supports financial planning and forecasting.
